# AiXWorkForce360 Backend

Backend for AiXWorkForce360 application using Node.js, TypeScript, Express, PostgreSQL, and Prisma/TypeORM.

## Folder Structure
See folder structure.md for detailed layout.

## Setup
1. Copy .env.example to .env and update values.
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

## Scripts
- `dev`: Start development server
- `build`: Compile TypeScript
- `start`: Run production build
- `migrate`: Run database migrations
- `seed`: Seed database
- `test`: Run tests
- `lint`: Run ESLint
- `format`: Run Prettier
